# react-native-news

> :warning: **This project is under DEVELOPMENT**!

## Starting 
- First install the dependences

```
npm i 
```
or

```
yarn 
```
- Running project on browser
```
npm run web
```
or

```
yarn web
```

> :warning: **If you are using browser**: The delete step will not work, it only works on mobile!
